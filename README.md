# AI Document Assistant (Chat with PDFs)

This project allows you to upload a PDF and ask questions about it using AI.

## 🚀 Features
- Upload PDF files
- Extract text from documents
- Ask questions using AI
- Get structured answers instantly

## 🛠 Tech Stack
- Python (Flask)
- OpenAI API
- PyPDF2
- HTML, CSS, JavaScript

## ⚙️ Setup Instructions

1. Clone the repo:https://github.com/opsai-studio/ai-document-assistant.git
2. Navigate into the folder: cd ai-document-assistant
3. Create virtual environment:python -m venv venv
                              venv\Scripts\activate
4. Install dependencies:pip install -r requirements.txt
5. Create a `.env` file: OPENAI_API_KEY=your_api_key_here
6. Run the app: python app.py
7.  Open browser: http://127.0.0.1:5000
